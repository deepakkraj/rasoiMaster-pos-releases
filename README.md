# Rasoi Master — Restaurant POS

**Rasoi Master** is a desktop **point-of-sale (POS)** application for restaurants and food outlets. It is designed for **day-to-day service**: quick billing, order handling, and back-office-friendly workflows, with an **offline-first** approach so counters can keep working when the network is slow or unavailable.

---

## What this app does

- **Billing & checkout** — Take orders, apply taxes (including GST-style line items where configured), discounts and coupons, and complete settlement smoothly.
- **Orders** — Manage live and held orders, search, and operational views suited to busy floors.
- **Kitchen & service** — Kitchen-stage and live views (where enabled) to coordinate preparation and service.
- **Settings & data** — Outlet setup, sync options, printing, security (e.g. PIN), and **in-app updates** when new versions are published.

Exact screens and options depend on your outlet configuration and the version you install.

## System requirements

| Platform | Requirement |
|----------|-------------|
| **Windows** | Windows **10 or later**, **64-bit (x64)**. Typical for shop and counter PCs. |
| **macOS** | **macOS** releases appear here as **`.dmg`** files when published—use the latest Release for your Mac (Intel vs Apple Silicon is called out in the file name when both are offered). |

You may need **administrator permission** on Windows to install or update the app.

---

## Installation (Windows)

1. Open **[Releases](https://github.com/deepakkraj/rasoiMaster-pos-releases/releases)**.
2. Under the latest release, download the Windows setup file (name like **`Rasoi-Master-0.1.0-Setup-x64.exe`** — the version number changes each release).
3. Double-click the installer. If Windows SmartScreen appears, choose **More info → Run anyway** only if you trust this download source.
4. Follow the installer steps, then start **Rasoi Master** from the **Start** menu or desktop shortcut.

## Installation (macOS)

1. Open **[Releases](https://github.com/deepakkraj/rasoiMaster-pos-releases/releases)**.
2. Download the **`.dmg`** for your Mac (file name includes **x64** or **arm64** when both builds exist).
3. Open the DMG, drag **Rasoi Master** into **Applications**, then eject the disk image.
4. On first open, if macOS blocks the app, use **System Settings → Privacy & Security** to allow it (steps may vary by macOS version).

---

## Updates

After installation, the app can **check for updates** from **Settings** (About / update section) and from the **update** control in the top bar. When a newer version is available on **Releases**, download and run the new installer (or follow any in-app **Install & restart** prompt). Restart the app when asked so the update finishes.
